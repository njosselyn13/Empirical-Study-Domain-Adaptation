Image Classification (MoCo Pretrained on ImageNet)
========================================================

We provide benchmarks of different finetune algorithms on `CUB-200-2011`_, `StanfordCars`_,
`Aircraft`_.

Those domain adaptation algorithms includes:

-  :ref:`BiTuning`

------------------------------------
